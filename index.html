Software Quality Assurance

Quality assurance is a systematic set of activities carried out to make sure that the delivered product is error-free and no glitch exists in the application. The absence of QA can lead to the risk of delivering the application with a glitch. And this is my strong area of expertise and I am continually learning and focusing on it to improve it.

Automation Testing

Automated tests perform the same steps precisely every time they are executed and never forget to record detailed results. Testers freed from repetitive manual tests have more time to create new automated software tests and deal with complex features. I have good experience of Automated testing with Selenium WebDriver and I have worked on understanding hands-on knowledge on new testing tools and API's like Katalon Studio, Jmeter, Selenium rest-assured.

API Testing

APIs, or application program interfaces, are vital tools for businesses in all industries. They are a means by which two different programs are able to communicate and testing of these API is vital part of any application since all the business knowledge in dependant on them, I am having good understanding of API and web-service level testing and very good grasp on HTTP Methods, HTTP status code and automated tools like Postman, SoapUI, Curl.

Software Installation and Deployment

It is very important to make sure that user does not face any difficulties while installing the software.For Enterprise applications (desktop or windows based), it always requires proper installation and configuration testing prior to product release to avoid risk of facing installation errors, miss configurations on customer environments. I am having experience in entire Installation and deployment scenarios and I posses a good understanding about DevOps.
